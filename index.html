FPGA Decryptor

ECE 287: Digital Systems Design - Fall 2011

Two FPGA's connected via ribbon cables. One of the boards is interfaced with a VGA monitor and PS/2 keyboard. Written in Verilog, VHDL, and Block Schematic.

The two FPGA's communicate using a defined protocol. The "master board" sends encrypted information to the "slave board" which cracks the encryption and displays the information on the monitor. The slave board takes input from the PS/2 keyboard and displays it on the monitor. Pressing enter sends the information to the master board.

Most solutions for the problem take about 3-4 minutes, this solution takes 0.209 seconds.

Krencrypt Password Manager

Free time - Spring 2012

This project was done for the purpose of managing my own personal passwords. I had been reading quite a bit about information security and entropy at the time and wanted a way to save secure passwords. Rather than have a file containing all of the passwords that could possibly stolen, I decided to make a program that would consistently generate unique passwords.
