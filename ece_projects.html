FPGA Decryptor

ECE 287: Digital Systems Design - Fall 2011

This project utilized two Altera DE2 Cyclone II FPGA's connected with ribbon cables, a VGA monitor, and a PS/2 keyboard. One FPGA was a "master board" that contained an encrypted message that contained a code. The other FPGA, the "slave board" sends a message to the master board requesting the message and the master board sends the packets and starts a timer. From there, the slave board decrypts the message and parses out the code. The code must then be sent back to the master board. Once the correct code has been received, the master board stops the timer and displays the time. There are three different levles that the slave board must go through and the timer keeps track of the total time taken.

This solution interfaced the slave board with a VGA monitor and PS/2 keyboard to create a command line that gathers and displays the necessary information. The slave board does all of the decryption and interaction with the master board automatically and goes through each level without the need for input from the user. Most groups' slave boards finished the task around 3-4 minutes, needing either manual input of the code, or even manual decryption. This solution finishes the task in about 0.209 seconds.

Lego Robot

EAS 102: Problem Solving and Design - Fall 2010

This project was the last stage in a series of projects in which teams must build and program Lego NXJ robots to carry out various tasks. This robot was designed to follow a monorail-like track and stop at black lines. Once at the black lines, the robot checks either side for blocks. If there is a red block, it is to move on to the next black line, but if there is a green block, the robot is to knock the block over. The robot should continue along the track indefinitely until there is a signal to stop.

My team's robot was the only one to execute the entire task perfectly. As an added bonus, the robot was programmed to do the entire task in pitch black darkness as well as daylight.
